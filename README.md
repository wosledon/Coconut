# Coconut

一款面向开发者和运维人员的单机 Web 工具，集 SSH 终端、SFTP 文件管理、服务器资源监控、AI 智能助手于一体。

## 快速开始

> 项目目前处于开发阶段，`src/` 和 `tests/` 尚待实现。

### 预览原型

```bash
open design/prototype.html
```

### 技术栈

- **后端**: ASP.NET Core 10 + EF Core + SQLite
- **前端**: React 19 + TypeScript + Tailwind CSS v4
- **实时通信**: SignalR
- **AI 流式**: SSE
