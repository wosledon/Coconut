# Coconut — SSH + SFTP 运维工具

## 产品概述

Coconut 是一款面向开发者和运维人员的单机 Web 工具，集 SSH 终端、SFTP 文件管理、服务器资源监控、AI 智能助手于一体。用户通过浏览器访问本地服务，统一管理多台服务器，借助 AI 提升日常运维效率。

---

## 核心目标

1. **统一入口** — 替代多个 Terminal/SFTP 窗口，一个界面管理所有服务器
2. **实时高效** — 终端操作零延迟感知，监控数据实时推送
3. **AI 赋能** — 自然语言转命令、日志智能分析、故障辅助诊断

---

| 角色       | 场景                                   |
| ---------- | -------------------------------------- |
| 后端开发   | 日常登录服务器查日志、改配置、重启服务 |

## 功能需求

### P0 — 核心体验

| 功能                       | 描述                                                                                                                                      |
| -------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------- |
| **连接管理**               | 增删改查 SSH 连接（主机/端口/用户/密码/密钥）；分组、标签、搜索；一键测试连通性                                                           |
| **SSH 终端（多会话 Tab）** | 基于 xterm.js 的浏览器内终端，通过 SignalR 与服务器 SSH Session 实时双向通信；支持同一连接创建多个终端会话 Tab，可切换/关闭，Tab 自动编号 |
| **SFTP 文件浏览**          | 远程目录树浏览、文件上传/下载/删除/重命名/新建目录                                                                                        |

### P1 — 重要功能

| 功能                          | 描述                                                                                                             |
| ----------------------------- | ---------------------------------------------------------------------------------------------------------------- |
| **资源监控（可收缩面板）**    | CPU / 内存 / 磁盘 / 网络 I/O 实时数据，每 2s SignalR 推送至顶部面板；面板可一键收缩，收缩后标题栏保留指标摘要    |
| **AI 提供商管理**             | 管理多个 AI 服务商（OpenAI / Anthropic / Ollama / 自定义兼容端点），API Key 加密存储，健康检查；集成在设置页面中 |
| **AI 会话聊天（可收缩面板）** | 基于 SSE 流式对话，保留历史消息；位于底部面板，右侧显示；面板可收缩，高度可拖拽调整                              |
| **AI 命令助手**               | 自然语言描述 → 生成 Shell 命令，可一键复制或发送到终端执行                                                       |

### P2 — 体验增强

| 功能                     | 描述                                                             |
| ------------------------ | ---------------------------------------------------------------- |
| **终端选中 → AI 解释**   | 终端中选中报错文本，通过 AI Chat 上下文功能发送给 AI 解释        |
| **一键诊断**             | AI 自动收集服务器信息并给出诊断报告                              |
| **AI 安全审计**          | AI 检查安全配置并出报告                                          |
| **Prompt 模板**          | 用户保存常用 AI 角色预设                                         |
| **设置页面（Tab 管理）** | 以左侧 Tab 导航管理：通用、终端、SSH、AI 提供商、数据、关于      |
| **面板高度拖拽**         | 底部面板（AI Chat/SFTP）与终端之间提供拖拽手柄，用户自由调整高度 |
| **面板收缩**             | 监控面板和底部面板均支持点击标题栏一键收缩/展开                  |

### P3 — 后续迭代

| 功能                       | 描述                     |
| -------------------------- | ------------------------ |
| 双面板 SFTP（本地 ↔ 远程） | 对比目录，拖拽上传       |
| 批量命令执行               | 选中多台服务器同时执行   |
| Docker 管理                | 查看容器、日志、执行命令 |
| 会话分享                   | 分享终端会话给他人       |
| 定时任务                   | 定时执行脚本，结果通知   |

## 技术方案

| 层       | 技术                                                |
| -------- | --------------------------------------------------- |
| 后端框架 | ASP.NET Core 10 (Controller-based Web API)          |
| 实时通信 | SignalR (Terminal / Monitor)                        |
| AI 流式  | Server-Sent Events (SSE)                            |
| 数据库   | SQLite + Entity Framework Core                      |
| SSH/SFTP | SSH.NET (Renci.SshNet)                              |
| 前端     | React 19 + TypeScript + Tailwind CSS v4             |
| 终端     | xterm.js                                            |
| 构建     | Vite                                                |
| 部署     | `dotnet run` 单机浏览器访问 `http://localhost:5000` |

## 数据模型

### SshConnection

| 字段                  | 类型     | 说明                   |

| Id                    | Guid     | 主键                   |
| Name                  | string   | 连接名称               |

| Port                  | int      | 端口，默认 22          |
| UserName              | string   | 登录用户               |
| AuthType              | enum     | Password / Key / Agent |
| EncryptedPassword     | string?  | DPAPI 加密存储         |
| KeyFingerprint        | string?  | 密钥指纹               |
| GroupName             | string?  | 分组                   |
| Tags                  | string?  | JSON 标签数组          |
| SortOrder             | int      | 排序号                 |
| CreatedAt / UpdatedAt | DateTime | 时间戳                 |

### AiProvider

| 字段                          | 类型            | 说明                                 |
| ----------------------------- | --------------- | ------------------------------------ |
| Id                            | Guid            | 主键                                 |
| Name                          | string          | 显示名称                             |
| ProviderType                  | enum            | OpenAI / Anthropic / Ollama / Custom |

| EncryptedApiKey               | string?         | DPAPI 加密                           |
| DefaultModel                  | string          | 默认模型                             |
| Temperature                   | double          | 默认 0.7                             |

| IsEnabled / IsDefault         | bool            | 状态标记                             |
| LastHealthCheckAt / IsHealthy | DateTime / bool | 健康检查                             |

### AiChatSession / AiChatMessage

| 字段                      | 类型    | 说明                    |
| ------------------------- | ------- | ----------------------- |
| SessionId                 | Guid    | 会话 ID                 |
| Title                     | string  | 会话标题                |

| SshConnectionId           | Guid?   | 关联服务器（可选）      |

| Message.Content           | string  | 消息内容                |
| Message.ContextSnapshot   | string? | 附加的服务器上下文 JSON |

## 非功能需求

- 单机工具，无需用户认证
- 页面启动即连 SQLite，无需额外配置数据库
- AI 流式响应首字延迟 < 2s
- 监控数据推送间隔 2s
- SSH 密码/Key 在传输和存储中加密（DPAPI）
- API Key 在数据库中加密存储（DPAPI）
- 面板高度可拖拽记忆
- 终端会话 Tab 可动态增删
